# Settings Information Architecture

## Sitemap

### General
- **Appearance & Language**: Theme, Language.
- **Startup & Tray**: Run at login, Minimize to tray.
- **Modes & Presets**: Active preset selection.
- **Power & Performance**: CPU usage limits, Battery saver.
- **Data Handling**: Telemetry, Crash reporting.

### Workspaces
- **Workspace Switcher**: Select active workspace.
- **Workspace Defaults**: Templates for new workspaces.
- **Import & Export**: Import/Export settings.

### Locations
- **Included Locations**: Folders to index.
- **Exclusions**: Folders to skip.
- **Policy**: Enforced restrictions (network drives, etc).

### Indexing
- **Status**: Current indexing state.
- **Change Detection**: Real-time vs Scheduled.
- **Performance**: Priority.
- **Content Limits**: Max file size.

### AI Models
- **Runtime & Hardware**: CPU/GPU selection.
- **Model Set**: Basic/Advanced models.
- **Quality Controls**: Confidence thresholds.

### Extraction
- **Documents**: OCR settings.
- **Images**: Analysis settings.
- **Code**: Source code indexing.
- **Archives**: Recursion depth.
- **Media**: Transcription.
- **Email & Exports**: Outlook integration.

### Organize & Automations
- **Organizing Mode**: Copy/Move/None.
- **Rules**: Auto-run rules.
- **Safety & Undo**: Confirmation prompts.
- **Review Queue**: Manual review settings.

### Tags & Taxonomy
- **Tagging Behavior**: Auto-tagging.
- **Vocabulary & Synonyms**: Custom terms.
- **Windows Metadata**: Write-back to file system.

### Partners
- **Directory**: Partner apps.
- **Matching Logic**: Similarity thresholds.
- **Policies**: Data sharing.

### Search
- **Search Mode**: Default search behavior.
- **Results & Previews**: UI options.
- **Ranking & Facets**: Algorithm selection.

### Windows Integration
- **Explorer Surfaces**: Context menu, Overlays.
- **Context Menu Actions**: Right-click options.
- **Windows Search Integration**: System search.

### Privacy & Security
- **Local Storage & Encryption**: DB Encryption.
- **Access Controls**: Authentication.
- **Sensitive Data Protections**: PII Masking.
- **Keys & Rotation**: Key management.

### Compliance & Audit
- **Audit Log**: Enable logging.
- **Retention & Legal Hold**: Data retention.
- **Policy Reports**: Compliance reporting.

### Notifications
- **Alerts & Banners**: UI notifications.
- **Quiet Hours**: Suppression schedules.

### Storage & Maintenance
- **Storage Locations**: Index path.
- **Cache Controls**: Cache limits.
- **Maintenance & Repair**: Auto-repair.
- **Backup & Restore**: Backup settings.

### Diagnostics
- **System Status**: Health check.
- **Performance Trace**: Tracing.
- **Export Diagnostics Bundle**: Debug info.
- **Advanced (Power User)**: Verbose logging.

### Admin
- **Policy Overview**: Enforcement status.
- **Restrictions**: List of locks.
- **Deployment & Updates**: Update channels.

### About
- **Version & Licenses**: App info.
- **Reset**: Factory reset.
